\  convert string to number, stopping at first non-digit 
:  >number        \  ud a u -- ud' a' u'   \ 6.1.0570

      over  c@   base  @  tuck  digit?  if     \  ud a u base n

         r>  0  d+   2swap

         2drop  exit 
      then 
      1  /string 
   repeat
